Elevator pitch

  • Docker is an opensource engine for containers
  • Container technology is more than a decade old. What’s different about Docker is that it makes containers easier to use
  • Promises to help simplify "cloud" / environment portability
  • Less resource
  • Very active FOSS project gain a lot of traction

Dude, isn't is just a VM?

  • Often compared to virtualization
  • virtual machine you’ll find a full operating system install with the associated overhead of virtualized device drivers, memory management, etc.
  • containers use and share the OS and device drivers of the host.
  • containers are therefore smaller than VMs, start up much faster, and have better performance
